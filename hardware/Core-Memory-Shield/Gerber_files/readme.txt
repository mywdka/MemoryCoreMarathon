These are the RS-274X gerber files for both the core board and the driver board.

The filename extensions are as follows:

GBL Bottom layer (copper)
GTL Top layer (copper)

GBS Bottom soldermask
GTS Top soldermask

GBO Bottom overlay (silkscreen)
GTO Top overlay (silkscreen)

TXT Excellon drill file	(2.4 with leading zero suppression)

GTP Top paste (used to specify stencil - necessary only for driver board)
